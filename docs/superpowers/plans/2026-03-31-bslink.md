# bslink Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a `bslink` script that interactively symlinks selected folders from `~/Github/backseat-docs` into `<cwd>/backseat-docs/`, with pre-selection of any existing links.

**Architecture:** Single bash script in `dotfiles/scripts/bslink`. Uses `gum choose --no-limit` for interactive multi-select. On each run, detects existing symlinks in `<cwd>/backseat-docs/`, pre-ticks them, then diffs old vs new selection to add/remove symlinks.

**Tech Stack:** bash, gum (charmbracelet/gum)

### Task 5: Final cleanup and stow

**Files:**
- Modify: `scripts/bslink` (review pass)

- [ ] **Step 1: Review the full script for correctness**

Read the complete `scripts/bslink` and verify:
- All sections flow in the correct order: guards → scan → detect existing → gum → diff → apply → cleanup
- `array_contains` is defined before it is called
- `set +e` / `set -e` wraps only the gum call

The final script should look like:

```bash
#!/usr/bin/env bash
# bslink — interactively symlink folders from backseat-docs into cwd
# Usage: bslink

set -euo pipefail

SOURCE_DIR="$HOME/Github/backseat-docs"
LINK_DIR="$(pwd)/backseat-docs"

if ! command -v gum &>/dev/null; then
    echo "bslink requires gum: https://github.com/charmbracelet/gum"
    exit 1
fi

if [[ ! -d "$SOURCE_DIR" ]]; then
    echo "Source directory not found: $SOURCE_DIR"
    exit 1
fi

mapfile -t all_folders < <(
    find "$SOURCE_DIR" -maxdepth 1 -mindepth 1 -type d -printf '%f\n' | sort
)

if [[ ${#all_folders[@]} -eq 0 ]]; then
    echo "No folders found in $SOURCE_DIR"
    exit 0
fi

existing=()
if [[ -d "$LINK_DIR" ]]; then
    while IFS= read -r -d '' link; do
        existing+=("$(basename "$link")")
    done < <(find "$LINK_DIR" -maxdepth 1 -mindepth 1 -type l -print0 2>/dev/null)
fi

selected_str=$(IFS=,; echo "${existing[*]}")

set +e
new_selection_raw=$(printf '%s\n' "${all_folders[@]}" | \
    gum choose --no-limit --selected="$selected_str" --header="Select backseat-docs folders to link:")
gum_exit=$?
set -e

if [[ $gum_exit -ne 0 ]]; then
    echo "Cancelled."
    exit 0
fi

mapfile -t new_selection <<< "$new_selection_raw"
if [[ ${#new_selection[@]} -eq 1 && -z "${new_selection[0]}" ]]; then
    new_selection=()
fi

array_contains() {
    local needle="$1"; shift
    local item
    for item in "$@"; do
        [[ "$item" == "$needle" ]] && return 0
    done
    return 1
}

for folder in "${existing[@]}"; do
    if ! array_contains "$folder" "${new_selection[@]+"${new_selection[@]}"}"; then
        rm "$LINK_DIR/$folder"
        echo "Removed: backseat-docs/$folder"
    fi
done

if [[ ${#new_selection[@]} -gt 0 ]]; then
    mkdir -p "$LINK_DIR"
    for folder in "${new_selection[@]}"; do
        target="$LINK_DIR/$folder"
        if [[ -e "$target" && ! -L "$target" ]]; then
            echo "Warning: $target exists and is not a symlink — skipping"
            continue
        fi
        if ! array_contains "$folder" "${existing[@]+"${existing[@]}"}"; then
            ln -s "$SOURCE_DIR/$folder" "$target"
            echo "Linked: backseat-docs/$folder"
        fi
    done
fi

if [[ -d "$LINK_DIR" ]] && [[ -z "$(ls -A "$LINK_DIR")" ]]; then
    rmdir "$LINK_DIR"
fi
```
